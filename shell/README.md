# Shell

一些shell的通用配置

- `bash.bashrc`
- `zsh.zshrc`
- `alias.sh` 命令别名
- `environment.sh` 环境变量
- `prompt.sh` `git-prompt.sh` 提示符
  - zsh默认显示git状态，bash默认不显示
  - `[Hostname Username path] (branch *=)`
  - `[$]:`
  - 禁用git提示符 `_git_prompt_disable`
  - 启用git提示符 `_git_prompt_enable`
- `gitconfig`
- `sshconfig`
- `tmux.conf`
- `vimrc`
